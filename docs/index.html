마음장부 개인정보 처리방침

시행일: 2026년 3월 22일

마음장부(이하 "앱")는 사용자의 개인정보를 소중히 여기며, 관련 법령에 따라 아래와 같이 개인정보 처리방침을 안내합니다.

1. 수집하는 개인정보

마음장부는 개인정보를 수집하지 않습니다.

  • 모든 데이터(경조사 기록, 이름, 금액 등)는 사용자의 기기 내에만 저장됩니다.
  • 외부 서버로 데이터를 전송하거나 저장하지 않습니다.
  • 회원가입, 로그인 기능이 없습니다.

2. 카메라 및 사진 접근

축의록 스캔 기능 사용 시 카메라 및 사진 라이브러리 접근 권한을 요청할 수 있습니다.

  • 촬영된 사진은 텍스트 인식(OCR) 처리 후 기기 내에서만 사용됩니다.
  • OCR 기능 사용 시 Google Cloud Vision API를 통해 이미지가 처리될 수 있으며, Google의 개인정보 처리방침이 적용됩니다.

3. 데이터 저장

  • 모든 데이터는 기기 내 로컬 데이터베이스(SQLite)에 저장됩니다.
  • 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.
  • 엑셀 내보내기/가져오기 기능은 사용자가 직접 파일을 관리합니다.

4. 제3자 제공

마음장부는 사용자의 데이터를 제3자에게 제공하지 않습니다.

5. 아동의 개인정보

마음장부는 만 14세 미만 아동의 개인정보를 수집하지 않습니다.

6. 개인정보 처리방침의 변경

본 방침이 변경되는 경우 앱 업데이트 또는 본 페이지를 통해 공지합니다.
